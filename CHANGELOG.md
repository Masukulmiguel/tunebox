### Added
- Splash screen com animação
- Autenticação (Email + Google)
- Tela principal com seções: Tendências, Novos Lançamentos, Top Artistas
- Player de áudio completo com controles
- Mini player na barra inferior
- Biblioteca: Favoritos, Downloads, Histórico, Playlists
- Pesquisa em tempo real
- Publicação de músicas
- Criar e gerir playlists
- Comentários em músicas
- Perfil do utilizador
- Perfil do artista
- Seguir/Deixar de seguir artistas
- Curtir/Remover curtida
- Partilha (WhatsApp, Facebook, Telegram)
- Design System completo (Core/Theme)
- Modo escuro e claro
- Animações suaves
- Responsive layout
- Supabase Backend (Auth, Database, Storage, Realtime)
- Hive para cache offline
- Firebase Analytics
- Firebase Crashlytics
- Firebase Cloud Messaging

- Riverpod para gestão de estado

- Design System reutilizável
